# WorkOS plugin marketplace

**New teammate? Start with the [Setup Guide](SETUP.md)** — the full 15-minute Day-1
walkthrough (folder, Team shortcut, plugin install, first run, and what's yours to
customize).

Install the WorkOS skill family (the AE daily driver) into Claude / Cowork:

1. **Settings → Customize → Plugins → Add marketplace** → paste `kylerw/workos-plugins` → Sync
2. Install the **workos** plugin
3. New chat: run `/workos-setup init my workspace` (first time) or `/workos-setup check my setup`

Updates ship as new versions — use the **Update** button on the plugin. No account or
special access needed; this repo is public and contains only built skill content.

## What's inside

| Skill | Does |
|---|---|
| `workos-setup` | Workspace init (brownfield-safe, additive-only) + `doctor` health checks |
| `workos-sync` | The daily driver: full sync / tidy passes + the day board (artifact + mobile snapshot) |
| `workos-next-steps` | Weekly next-steps sweep, locked-format Salesforce lines, manager email |

## Provenance

Built content only, published from the private WorkOS engine repo on each release tag
(`scripts/publish-plugins.sh`). Contract-enforced: **no customer data ever** (all examples
fictional), no personal values (everything resolves through per-user config generated by
`workos-setup`). Current version: see `plugins/workos/.claude-plugin/plugin.json`.
